# 🌡️ Temperature Converter Pro

A modern desktop application built using **Python** and **CustomTkinter** that converts temperatures between Celsius, Fahrenheit, and Kelvin with a professional user interface and additional utility features.

---

## 📌 Project Overview

Temperature Converter Pro is a user-friendly temperature conversion application developed as part of my **Software Development Internship at SkillCraft Technology**.

The application allows users to quickly convert temperature values between different units while maintaining a history of conversions and providing export options.

---

## ✨ Features

- 🌡️ Convert temperatures between:
  - Celsius (°C)
  - Fahrenheit (°F)
  - Kelvin (K)

- 🎨 Modern Rose & Black UI Design
- 📜 Conversion History Tracking
- 💾 Save History as TXT File
- 📊 Export History to CSV File
- 🌙 Dark/Light Mode Toggle
- ⌚ Live Date & Time Display
- ℹ️ About Section
- 🚀 Splash Screen on Startup
- ⌨️ Enter Key Support
- 🖥️ Responsive Desktop Interface

## 🚀 Installation

### Step 1: Clone the Repository

```bash
git clone https://github.com/Rash6906/SCT_SD_1.git
```

### Step 2: Navigate to Project Folder

```bash
cd SCT_SD_1
```

### Step 3: Install Required Package

```bash
pip install customtkinter
```

### Step 4: Run the Application

```bash
python Temperaturepro.py
```
